# Bitcoin Price Predictor

A machine learning web application that predicts Bitcoin prices and gives buy/sell suggestions.

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML
- CSS
- JavaScript
- Chart.js

## Features
- Predicts Bitcoin price using ML model
- Shows Buy/Sell suggestion
- Interactive price visualization chart
- Simple web dashboard

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run backend server

cd server_api
python app.py

3. Open frontend

Open file:///C:/Users/Varshitha/Documents/btc-price-predictor/client_dashboard/index.html

Enter price values and click Predict.
